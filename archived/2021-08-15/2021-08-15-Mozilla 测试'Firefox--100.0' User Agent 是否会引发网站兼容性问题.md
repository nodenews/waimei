[Mozilla 测试'Firefox/100.0' User Agent 是否会引发网站兼容性问题](https://www.solidot.org/story?sid=68544)
------

<div>
浏览器使用用户代理字符串（User Agent String）告诉网站它的相关软件消息，如名称、版本号和技术。网站根据这些信息修改不同功能的响应。每当发布新版本，用户代理字符串里的版本号也会更新。到目前为止，Mozilla Firefox 的版本号一直是两位数，计划在 2022 年 3 月发布的 Firefox Nightly 的版本号将首次使用三位数，开发者<a href="https://bugzilla.mozilla.org/show_bug.cgi?id=1719070">想知道</a>三位数版本号的用户代理字符串是否会破坏网站的功能，引发兼容性问题。他们开始<a href="https://www.bleepingcomputer.com/news/software/mozilla-tests-if-firefox-1000-user-agent-breaks-websites/" target="_blank">提前进行测试</a>，测试使用的用户代理是“Mozilla/5.0 (Windows NT 10.0; rv:100.0) Gecko/20100101 Firefox/100.0”。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
