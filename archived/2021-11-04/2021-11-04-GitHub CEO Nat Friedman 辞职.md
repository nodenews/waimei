[GitHub CEO Nat Friedman 辞职](https://www.solidot.org/story?sid=69503)
------

<div>
GitHub CEO Nat Friedman <a href="https://github.blog/2021-11-03-thank-you-github/">宣布辞职</a>，他将成为荣誉主席，继续担任微软和 GitHub 的顾问。微软是在 2018 年收购 GitHub，选择 Xamarin 前 CEO Nat Friedman 担任收购后独立运营的子公司 CEO。GitHub 的首席产品官 Thomas Dohmke 将于 11 月 15 日接任 CEO，Dohmke 是在 2009 年注册为 GitHub 用户，当时的 GitHub 仅成立一年。他曾是应用测试创业公司 HockeyApp 的联合创始人兼 CEO，微软在 2014 年收购 HockeyApp，在收购 GitHub 之后他转到了 GitHub。微软云计算和人工智能集团执行副总裁 Scott Guthrie <a href="https://slashdot.org/story/21/11/03/2229225/microsofts-github-ceo-nat-friedman-is-stepping-down" target="_blank">称</a> Dohmke 还领导了对两家创业公司  Npm 和 Semmle 收购。
</div>
