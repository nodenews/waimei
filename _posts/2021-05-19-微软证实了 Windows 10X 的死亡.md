[微软证实了 Windows 10X 的死亡](https://www.solidot.org/story?sid=67817)
------

<div>
微软<a href="https://blogs.windows.com/windowsexperience/2021/05/18/how-to-get-the-windows-10-may-2021-update/">正式承认</a>它不会发布 Windows 10X。Windows 10X 是微软对 Google Chrome OS 的回应，设计作为一种简化版的 Windows 10 提供给教育和一线工人市场，它原计划在 2021 年推出。但现在微软表示，它认为 Windows 10X 的技术能在更多方面有用，能比最初预计的服务于更多用户，它将把 Windows 10X 的关键技术整合到 Windows 等产品中，部分已经整合到预览版中，如  Microsoft Defender Application Guard 引入的新应用容器技术，增强的  Voice Typing 体验等等。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
