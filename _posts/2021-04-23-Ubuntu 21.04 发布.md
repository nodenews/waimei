[Ubuntu 21.04 发布](https://www.solidot.org/story?sid=67586)
------

<div>
Canonical 正式<a href="https://ubuntu.com//blog/ubuntu-21-04-is-here" target="_blank">释出了</a> Ubuntu <a href="https://ubuntu.com/download">21.04</a>。主要新特性包括：原生微软 Active Directory 集成，Ubuntu 机器可以在安装时加入 Active Directory（AD）域以进行集中配置，AD 管理员现在可以管理 Ubuntu 工作站，从而简化了对公司策略的合规性；默认使用 Wayland；Snap 版 Flutter SDK 构建集成；重新设计深色主题 Yaru，等等。软件方面的更新包括：5.11 Linux 内核，gcc 10.3.0，openssl 1.1.1，Openstack Wallaby、Rails 6、 QEMU 5.2、libvirt 7.0、Kubernetes 等。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
