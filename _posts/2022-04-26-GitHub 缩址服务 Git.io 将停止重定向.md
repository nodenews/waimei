[GitHub 缩址服务 Git.io 将停止重定向](https://www.solidot.org/story?sid=71361)
------

<div>
GitHub <a href="https://github.blog/changelog/2022-04-25-git-io-deprecation/">宣布</a>其缩址服务 Git.io 将于 4 月 29 日停止重定向。这意味着在这个日期之后 Git.io 缩址将找不到源网址。GitHub 建议使用 Git.io 的开发者立即更新其链接。缩址服务一度非常受欢迎，方便于在社交网络分享网址，但依赖于使用率不高的缩址服务会导致服务停止之后这些缩址成为了死链接。GitHub 的缩址服务 Git.io 因使用率不高，于今年 1 月进入只读模式，4 月 29 日终止服务。
</div>
