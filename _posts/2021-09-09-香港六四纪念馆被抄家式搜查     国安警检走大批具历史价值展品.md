[香港六四纪念馆被抄家式搜查     国安警检走大批具历史价值展品](https://www.rfa.org/mandarin/yataibaodao/gangtai/ec-09092021072247.html)
------

<div>
<p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">香港国安处以搜证为名，进入支联会创立的六四纪念馆，带走大批馆内展出的六四物品，警员搜查完毕后还把大门锁上。天安门母亲发起人之一张先玲表示，质疑警方的行动，是否要抹走历史，隐瞒真相。</span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">香港警方国安处周四（</span><span style="font-size: 11.0pt; font-family: SimSun; color: black;">9<span lang="ZH-CN">日）中午前，带同在前一天被捕的支联会常委梁锦威，到在旺角的六四纪念馆搜查。</span></span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">警方在六四纪念馆逗留大约两个多小时，大批警员在大厦门口把守，架起封锁线，把大批存放在纪念馆的物品，包括民主女神像、身穿“平反六四、薪火相传”黑衫的司徒华展板，以及</span><span style="font-size: 11.0pt; font-family: SimSun; color: black;">20<span lang="ZH-CN">多箱放在胶盒内的书本和报章等物品，用货车送走。</span></span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><b><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">六四纪念馆大门被警方换锁犹如大陆式 “查封”</span></b></span></p><p style="margin: 12.0pt 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">已辞任支联会常委的蔡耀昌到现场了解情况，他在警方撤离后，尝试返回六四纪念馆检查。他接受本台访问时表示，纪念馆闭路电视被人破坏，大门的门锁也被更换，所以现在无法知道，警方在馆内做过什么事情。</span></span></p><p style="margin: 12.0pt 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">蔡耀昌：”见到最少在纪念馆外面楼层，在入口地方的闭路电视被拆除，有理由相信入口或在馆内的闭路电视也被破坏。估计门锁被拆除，再由警方装新的锁，在门口贴了警署的联络电话，支联会的人员如果需要开锁进入单位，要与警署联络。</span></span></p><p style="margin: 12.0pt 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">蔡耀昌表示，无法估计，纪念馆内有多少物品被警方带走，他担心，部分展品历史悠久，在运送的过程中，可能会被破坏。</span></span></p><p style="margin: 12.0pt 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">蔡耀昌：”当然是有这个担忧，一些的历史资料、当时的书籍、刊物、甚至是单张，有一些是原件，经历那么多年，比较容易受到损害，不论是很有价值或是有部分价值都好，都是历史资料，部分是原件，现在难以评估，如果真的有一些物品被拿走和被破坏，我们相信香港市民都非常不愿意看到和难以接受。”</span></span></p><p style="margin: 12.0pt 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">他重申，没有看到纪念馆内有任何物品，属于违法，但被警方拿走，认为警方需要向市民大众及支联会作详细解释，行动的目的、是否有足够法律依据等，支联会也应该就警方行动的合法性，谘询法律意见。</span></span></p><p style="margin: 12.0pt 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><b><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">张先玲质疑警方的行动是否要抹走历史隐瞒六四真相</span></b></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">天安门母亲发起人之一张先玲在</span><span style="font-size: 11.0pt; font-family: SimSun; color: black;">8<span lang="ZH-CN">年前，把儿子王楠的遗物，捐赠予六四纪念馆展出。她表示，不清楚六四纪念馆内有什么物品被带走，质疑警方的行动，是否要抹走历史，隐瞒真相。</span></span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">张先玲：“六四事件是光天化日之下，政府动用正规军队杀无辜的市民，这是一种罪行、是国家的刑事犯罪，他现在将这些东西拿走，是否想隐瞒事实呢？但事实是无法被隐瞒的，还有那么多遇难者家属在世，每个经历过六四事件的人都不会忘记六四杀人的事情，想把事情隐瞒是不可能的，想把维园晚会或六四纪念馆破坏，六四就会被人忘记，我相信不会的。”</span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">天安门母亲发言人尤维洁，亦对香港警方检走六四纪念馆的物品感到遗憾和愤慨，她反问，透过六四纪念馆，保存和展示亲人遗物，如何危害国家安全。</span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">尤维洁：”遗物的话，不涉及香港国家安全法，因为这都是实质存在的，是当年六四惨案每个家庭失去亲人的这种悲痛，孩子和亲人的遗物捐给香港六四纪念馆，也希望世人不要忘记六四惨案，我们用这种方式让子孩后世和世人，永远记着六四惨案，都会被认为是危害国家安全吗？”</span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><b><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">支联会再多一名常委被捕</span></b></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">另外，警方在周三</span><span style="font-size: 11.0pt; font-family: SimSun; color: black;">(9<span lang="ZH-CN">月</span>8<span lang="ZH-CN">日</span>)<span lang="ZH-CN">清晨拘捕包括支联会副主席邹幸彤及</span>3<span lang="ZH-CN">名常委，指他们涉及《港区国安法》第</span>43<span lang="ZH-CN">条下，有关“没有遵从通知规定提供资料罪”后，警方周四再拘捕常委徐汉光。</span></span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">美英高层官员在社交网站发文批评拘捕行动，中国外交部驻港特派员公署强烈回应，批评西方政客说三道四，表示强烈不满和坚决反对，要求外国政客，立即停止为香港反中乱港分子撑腰打气、干预香港事务和法治，干涉中国内政，否则必将遭到中国人民的坚决回击。</span></span></p><p style="margin: 0cm 0cm 12.0pt 0cm;"><span style="white-space: pre-wrap;"><span lang="ZH-CN" style="font-size: 11.0pt; font-family: SimSun; color: black;">记者：陈妙玲 责编 胡力汉 许书婷</span></span></p>
